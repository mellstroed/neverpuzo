# Unfl4gg3d Changelog

## Версия 1.2.0 - Исправление багов, добавление новых функций, удаление ненужных функций
**Дата выхода:** Апрель 2026

### Возможности
* Поддержка множества читов для CS:GO
* Автоматическая загрузка и инжект система
* Переключение языка (RU/EN)
* Темы оформления ( можно выбрать самому, так как есть цветовая палитра и HEX код )
* Выбор шрифтов
* Можно выбрать белую тему, тема сразу ставится в зависимости от темы на компьютере, темы сохраняются


### Поддерживаемые читы
* **NeverLose** - Полная поддержка сервера, фикс луашек, конфигов
* **Ev0lve** - Старый закрывшийся чит
* **Primordial** - Красивый чит
* **Legendware** - Легендарная Паста
* **GameSense** - 40 бачей инвайт
* **OneTap** - воньтап
* **Fatality (siniy)** - синий фаталити, есть брат красный, розовый
* **bob** - скэбоб.вип
* **FURRYWARE** - фурятина
* **Havoc** - дэфэнсив флэки
* **sosality** - брат синего, красного фаталити
* **vivo 2.5** - виво х300 про ( вейв )
* **spermaware** - паста рейза
* **naval** - алексей навальный
