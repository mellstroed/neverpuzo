# Unfl4gg3d — История изменений

## Версия 1.3.0- Обновление интерфейса
Дата выхода: Апрель 2026

### Новое
- Тёмная / Светлая тема с автоопределением системной темы Windows
- Цветовая палитра с полным HSV-пикером и вводом HEX
- Выбор шрифта: Segoe UI, Arturito, Moyenage, Pixeloid, Google Sans
- Снег с цветом снежинок под тему (белые / чёрные)
- Перетаскивание читов с сохранением порядка
- История инжектов: последний инжект + счётчик на карточке
- Переключатель языка RU / EN с мгновенной перезагрузкой
- Анимация заголовка с эффектом печатной машинки и глитчем
- Наложение CRT-сканлайнов
- Плавная прокрутка во всех панелях
- Автоматическое изменение логотипа NEVERLOSE на "unfl4gg3d" после инжекта
- Встроенный DLL инжектор (работает если внешний инжектор отсутствует)

### Исправлено
- Кастомный акцент сохраняется при смене темы
- Тема автоматически определяется из реестра Windows при первом запуске
- Продукты корректно перезагружаются при смене языка
